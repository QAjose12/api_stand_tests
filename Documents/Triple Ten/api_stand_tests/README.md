#  api_stand_tests

Proyecto de pruebas automatizadas para interactuar con la API de **Urban Grocers**, utilizando la librería **requests** en Python.  
Aquí se envían solicitudes HTTP a los endpoints documentados y se validan las respuestas mediante pruebas automatizadas.

---

##  Objetivo del Proyecto

Este proyecto tiene como propósito:

- Enviar solicitudes a la API de Urban Grocers.
- Validar respuestas utilizando pruebas automatizadas.
- Practicar el uso de `requests`, buenas prácticas de testing y manejo de datos.
- Implementar pruebas positivas y negativas de los endpoints disponibles.

---

##  Tecnologías

- **Python 3.x**
- **requests**
- **pytest**
- PyCharm (entorno recomendado)
